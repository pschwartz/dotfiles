# Dotfile management with symlinks using `stow`

Dependencies:

```
stow
zsh
starship
eza
zoxide
python-pygments
neovim
tmux
zellij
ghostty
```

Currently all installed via `extra` or `aur` on Arch Linux

==========================================

To install use the following:

```
./setup.sh
```

or

```
stow .
```
